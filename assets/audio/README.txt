Здесь хранятся заранее созданные MP3 для двух голосов:

  gabrijela/ — София, мягкий женский сербский голос
  srecko/    — Никола, спокойный мужской сербский голос

Названия папок сохранены от предыдущей версии для совместимости.
На Windows запустите start-windows.bat.
Для короткой проверки без полной генерации используйте test-voices-only.bat.
Ключ Azure, Microsoft-аккаунт и банковская карта не нужны.
Интернет используется только во время однократного создания MP3.
После этого приложение воспроизводит звук локально.
