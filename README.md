# Ćao! — сербский A1

Статическое веб-приложение для изучения сербского с русского языка.

## Что входит

- полный маршрут A1: 39 тем и короткие последовательные уроки;
- алфавит и сербская латиница/кириллица;
- базовая лексика для быта, работы, жилья, поездок и документов;
- мини-грамматика, аудирование, сборка фраз и ввод слов;
- быстрый адаптивный тест на 6–10 вопросов;
- полный тест на 24 вопроса;
- интервальное повторение слов;
- два локальных голосовых пакета;
- сохранение прогресса в браузере.

Авторизация пока не подключена. Прогресс хранится в `localStorage` и позже может быть перенесён в Supabase.

## Обновление звуков без повторной генерации

1. Скопируйте `cao-audio-generator.exe` из предыдущей локальной сборки в корень этой папки.
2. Сохраните существующие папки:
   - `assets/audio/gabrijela/`
   - `assets/audio/srecko/`
3. Запустите `generate-only-missing-audio.bat`.

Скрипт сначала объединяет старый аудиоманифест с новым курсом. Затем он проверяет наличие каждого MP3 у обоих голосов и передаёт генератору только недостающие записи. Уже созданные файлы не удаляются и не создаются заново.

Если генерация оборвалась, просто запустите BAT ещё раз: завершённые записи будут пропущены.

## Публикация на GitHub Pages

После генерации загрузите в репозиторий сайт вместе с папкой `assets/audio`. В настройках репозитория включите Pages из ветки `main`, папка `/ (root)`.

Для работы сайта на хостинге EXE, BAT и папка `tools` не обязательны. Они нужны только локально для дальнейшего расширения курса.

## Запуск локально

Можно открыть `index.html` напрямую или запустить любой простой локальный HTTP-сервер.
