# Ćao! — тренажёр сербского языка

Адаптивный фронтенд-прототип для изучения сербского с абсолютного нуля. Сборщик, npm и бэкенд для запуска не нужны.

## Запуск

На Windows можно открыть `start-windows.bat`. На macOS — `start-mac.command`.

Также можно открыть `index.html` напрямую или запустить локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Что реализовано

- 8 разделов и 23 последовательных урока;
- отдельный стартовый раздел со всеми 30 буквами сербского алфавита;
- новые слова сначала показываются на обучающей карточке с переводом и озвучкой;
- в первых разделах нет ввода с клавиатуры и ошибки не отнимают сердца;
- выбор ответа, обратный перевод, аудирование, ввод слова и сборка фраз;
- пунктирные подсказки: тап или клик по сербскому слову открывает перевод;
- сербская латиница и кириллица;
- выбор системного голоса и режимов произношения: мягкий, обычный, медленный;
- пять растровых PNG-персонажей со случайными репликами;
- XP, сердца, ежедневная цель и серия;
- интервальное повторение слов;
- словарь с поиском и фильтрами;
- сохранение прогресса в `localStorage`;
- адаптивная мобильная навигация.

## Озвучка

Приложение использует голоса, установленные в операционной системе и доступные браузеру через SpeechSynthesis. Сербский голос выбирается первым, затем хорватский или боснийский. Конкретный набор вариантов зависит от устройства пользователя.

## Файлы

- `index.html` — разметка;
- `styles.css` — дизайн и адаптив;
- `app.js` — учебные данные, задания, подсказки, голоса и прогресс;
- `assets/characters/` — растровые персонажи PNG.
