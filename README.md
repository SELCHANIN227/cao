# Ćao! — тренажёр сербского языка

Готовый адаптивный фронтенд-прототип без сборки и внешнего бэкенда.

## Запуск

Откройте `index.html` в браузере или запустите простой локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Что реализовано

- оригинальный интерфейс в формате игрового языкового тренажёра;
- 6 разделов и 18 уроков;
- типы заданий: выбор ответа, обратный перевод, ввод слова, аудирование, сборка фразы;
- латиница и сербская кириллица;
- озвучка через SpeechSynthesis (`sr-RS`, если голос доступен в системе);
- случайные SVG-персонажи и реплики;
- XP, сердца, ежедневная цель и серия;
- интервальное повторение слов;
- словарь с поиском и фильтрами;
- сохранение прогресса в `localStorage`;
- адаптивная мобильная навигация.

## Файлы

- `index.html` — разметка;
- `styles.css` — дизайн и адаптив;
- `app.js` — уроки, данные, логика прогресса и интервального повторения.
